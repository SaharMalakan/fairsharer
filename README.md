## How to use fairsharer

Short description of package:
created an algorithm that fairshares. In each iteration, the largest value gives a fraction of itself to both neighboring elements.
examples: 
fair_sharer([0, 1000, 800, 0], 1) -- [100, 800, 900, 0]
fair_sharer([0, 1000, 800, 0], 2) -- [100, 890, 720, 90]

The project setup is documented in [project_setup.md](project_setup.md). Feel free to remove this document (and/or the link to this document) if you don't need it.

## Installation

To install fairsharer from GitHub repository, do:

```console
git clone https://github.com/SaharMalakan/fairsharer.git
cd fairsharer
python -m pip install .
# or on mac: python3 -m pip install .
```
